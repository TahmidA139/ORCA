FLOWCHARTS: ORCA – ORF Recognition and Comparative Annotator
=============================================================

These flowcharts describe the program's logic using text-based diagrams.
They are intended to be converted into graphical flowcharts for the final
presentation.

FLOWCHART 3: Six-Frame ORF Scanner
-------------------------------------

    [Start: receive cleaned DNA sequence, start codons, min length]
       |
       v
    [Compute reverse complement of sequence]
       |
       v
    [For each frame in [0, 1, 2]:]
       |
       v
    [Slice sequence starting at frame offset]
    [Reshape into array of 3-character codons using NumPy]
       |
       v
    [Build boolean start-codon mask]
    [Mark positions matching any requested start codon]
       |
       v
    [For each start codon position in mask:]
       |
       v
    [Find next in-frame stop codon (TAA, TAG, TGA)]
       |
       v
    [Stop codon found?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF is incomplete]
      |
      v
    [Calculate ORF length in nucleotides]
       |
       v
    [Length >= --min-length?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF too short]
      |
      v
    [Convert coordinates to forward-strand positions]
       |
       v
    [Is this the longest ORF for this stop codon?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — shorter ORF already recorded]
      |
      v
    [Record ORF: strand, frame, start, end, length, start codon]
       |
       v
    [Repeat for reverse complement strand]
       |
       v
    [Return flat list of all ORF records]
