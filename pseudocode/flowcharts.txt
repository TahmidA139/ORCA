FLOWCHART 3: Six-Frame ORF Scanner
-------------------------------------

    [Start: receive sequence, start codons, min length]
       |
       v
    [Compute reverse complement of sequence]
       |
       v
    [For each frame in [0, 1, 2]:]
       |
       v
    [Slice sequence at frame offset]
    [Reshape into codon array using NumPy]
       |
       v
    [Build boolean start-codon mask]
       |
       v
    [For each start codon position in mask:]
       |
       v
    [Find next in-frame stop codon]
       |
       v
    [Stop codon found?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF is incomplete]
      |
      v
    [Calculate ORF length in nucleotides]
       |
       v
    [Length >= min length?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF too short]
      |
      v
    [Convert coordinates to forward-strand positions]
       |
       v
    [Longest ORF for this stop codon?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — shorter ORF already recorded]
      |
      v
    [Record ORF: strand, frame, start, end, length, start codon]
       |
       v
    [Repeat for remaining frames and reverse complement strand]
       |
       v
    [Return flat list of all ORF records]
